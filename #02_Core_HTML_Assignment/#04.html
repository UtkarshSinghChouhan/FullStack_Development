HTML Tags List

  1. <h1>:

    This is a heading tag used for the main title or header of a page. It represents the highest level of importance.

  2. <p>:

    The <p> tag is used to define paragraphs of text, separating and formatting blocks of content.

  3. <a>:

    The <a> tag creates hyperlinks to other web pages or resources using the href attribute.

  4. <img>:

    The <img> tag embeds images in a web page. It requires the src attribute to specify the image URL.

  5. <ul>:

    <ul> is used to create unordered (bulleted) lists. It contains one or more <li> (list item) elements.

  6. <ol>:

    <ol> creates ordered (numbered) lists. It also contains one or more <li> elements.

  7. <input>:

    <input> creates various types of form input fields like text boxes, radio buttons, checkboxes, etc.

  8. <div>:

    <div> is a generic container for grouping and styling elements, often used in CSS layouts.
